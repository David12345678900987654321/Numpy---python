## Exercícios com NumPy em Python

Este repositório reúne exercícios desenvolvidos com a biblioteca NumPy, utilizando Python para praticar operações matemáticas, manipulação de arrays e processamento de dados numéricos.

Os exercícios abordam conceitos fundamentais da computação numérica e análise de dados, explorando recursos que tornam o processamento de informações mais rápido e eficiente.

### Conteúdos praticados
- Criação e manipulação de arrays
- Operações matemáticas e estatísticas
- Indexação e fatiamento de dados
- Tratamento de dados numéricos

### Tecnologias utilizadas
- Python
- NumPy
